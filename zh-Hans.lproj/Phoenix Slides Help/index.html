Phoenix Slides 帮助

简介

Phoenix Slides 是一个图片浏览及查看工具，其功能包括全屏幻灯片、 无损 JPEG文件转换、和EXIF 信息显示。此应用程序需要 macOS 10.13 或更高版本。

使用方法

你可以用窗口顶部的文件夹浏览表进入某个文件夹， 也可以直接把文件夹或文件夹里面的任何文件拖到窗口上。 若要播放幻灯片，你可以按“幻灯片”一下，或按一个缩略图两下。

文件夹浏览表

除了用鼠标按，你也可以用键盘打字来进入一个文件夹。

文件夹浏览表的第一栏列出目前所挂载的磁盘和卷宗。 当卷宗被挂载或退出，或者目前所选的文件夹被移动或重新命名时， 文件夹浏览表会自动更新。在其他情形下，你可以按一下一个文件夹的名称， 以更新其内容。

你可以按一下“子文件夹”来查找文件夹里的所有图档,包括子文件夹里图档。 你可以用这个功能查看应用程序套件内容的图像，很好玩。

缩略图

当你进入一个文件夹时，Phoenix Slides会先查找该文件夹里的图档， 然后开始生成缩略图。缩略图将会缓存于记忆体，退出应用程序时便会被丢弃。

选择图像

你可以用鼠标选择图像。如果刚好选了一个图像，你也可以用箭头键选择图像。

拖动图像

你可以将图像拖动到另外一个应用程序，让该应用程序打开文件， 或拖动到Finder以复制文件。

幻灯片

开始幻灯片时：

  • 若无所选项目，播放所有图像的幻灯片。
  • 若选了一个项目，从所选的图像开始播放所有图像的幻灯片。
  • 若选了数个项目，播放所选图像的幻灯片。

幻灯片播放时

请用键盘控制幻灯片。请打'h'或'?'键显示简单的说明。

请打1-9键幻灯片自动前进(每1-9秒)。打空白键来暂停；返回或打'0'键停止。

JPEG 转换

Phoenix Slides用jpegtran(包EXIF patch)来执行无损 JPEG 转换。

删除无法转换的边缘数据块 - 无损 JPEG旋转不一定是完美的。 如果图像的大小不是iMCU数据块大小的倍数(通常是8或16)， 侧边的数据块就无法转换了。你可以通过高级选项删除这些数据块。 当然这个动作无法撤销。详情请参阅jpegtran说明:

jpegtran's default behavior when transforming an odd-size image is designed
to preserve exact reversibility and mathematical consistency of the
transformation set.... Transpose is able to flip the entire image
area. Horizontal mirroring leaves any partial iMCU column at the right edge
untouched, but is able to flip all rows of the image. Similarly, vertical
mirroring leaves any partial iMCU row at the bottom edge untouched, but is
able to flip all columns. The other transforms can be built up as sequences
of transpose and flip operations; for consistency, their actions on edge
pixels are defined to be the same as the end result of the corresponding
transpose-and-flip sequence.

图像分组/分类

幻灯片播放时，你可以打F2-F12键将图像放进一个分组里， 然后在浏览窗口里再打F2-F12，查看一个分组内的图像。 你也可以用⌘F2-F12 来切换图像的分组类别。 这个功能很方便，你可以先选择很多图像，然后一起把它们旋转或执行其他动作。

Phoenix Slides的故事

有一天，马教授带了几张CD到办公室，CD上有他在泰国拍的照片。 “我们来看这些照片吧”他说。我们便在办公室的Mac里找适合的幻灯片应用程序， 不过找不到我们喜欢的。我们只好用苹果电脑预览应用程序。可是它非常慢， 而且用全屏的时候无法旋转图像！所以我打算寫我自己的应用程序， Phoenix Slides就这樣生成了。

我以前很喜欢用Caffeine Software的Curator应用程序，可惜它已经不再开发。 它也有其他的小毛病，例如，你要等到它生成缩略图才能看幻灯片。我觉得这樣太不合理了。 当然，Phoenix Slides没有这樣的毛病。

为什麼Phoenix呢？除了它的神話意义，它也是旧金山市旗的标誌(我住在旧金山)。 我也很喜欢Fawkes(在哈利波特里它是Dumbledore的凤凰).
